Privacy Policy for CodeCue Chrome Extension

CodeCue does not collect, store, or share any personal information from users.

The extension only accesses the content of the currently open LeetCode problem page to extract the question and send it to a secure backend proxy. This proxy interacts with a language model (LLM) API to return hints, pseudocode, and solutions.

  • No personal data is collected
  • No user activity is tracked
  • No data is shared with third parties
